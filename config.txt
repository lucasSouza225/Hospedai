frontend

1- npm create vite@latest <NOME DO PROJETO>
    Depois:
        Escolha React
        Escolha JavaScript ou TypeScript

    Em seguida:
        cd frontend
        npm install
        npm run dev

2- npm install tailwindcss @tailwindcss/vite
    2.1- npm install -D prettier prettier-plugin-tailwindcss
    2.2- depos por esse plugin logo no começo do "packge.json" :

    "prettier": {
    "plugins": [
      "prettier-plugin-tailwindcss"
    ]
  },

backend

1- npm init -y

  1.1- adicionar no "packge.json" ----- "type": "module",

2- npm i express

3- criar um aquivo index.js na pasta backend
  3.1- substuir o script no packge.json por:
    "scripts": {
    "dev": "node --watch ./index.js"
  }

4- npm i dotenv
  4.1 criar um aquivo na pasta backend com o nome .env (para rodar suas  próprias variaveis no backend)

5- npm install bcryptjs (para criptar a senha)
